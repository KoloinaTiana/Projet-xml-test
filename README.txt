Projet Restaurant par Koloina Rasolofoson et Maxime Soulié.

Fichier xml: restaurantCatalogue.xml
Fichier de validation avec XML schema: restaurantCatalogue.xsd

Fonctionnalités de l'application:
	- Affichage liste des plats dans un tableau
	- Recherche XQuery
	- Recherche XPath
	- Exportation en HTML avec la transformation en XSLT + CSS
	- Ajout de nouveau plat
	- Suppression de plat
